# Quant Trading Strategy Project

## 📊 Overview
This project implements a Moving Average Crossover trading strategy using Python.

## 🔧 Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- yfinance

## 📈 Strategy
- Buy when SMA50 > SMA200
- Sell when SMA50 < SMA200

## 💸 Backtesting
- Compared strategy performance with Buy & Hold
- Evaluated profit/loss

## 📌 Conclusion
The strategy underperformed the market, highlighting the importance of optimization and risk management.
